# Sound Files for Lifecycle Bot

## Required Files:
- `woohoo.mp3` - Plays on BUY events (Homer Simpson "Woohoo!")
- `awesome.mp3` - Plays on BLOCK/SAFETY events (Peter Griffin "Awesome")

## How to Replace:
1. Download your preferred sound clips (MP3 format)
2. Rename them to `woohoo.mp3` and `awesome.mp3`
3. Replace the placeholder files in this folder
4. Rebuild the APK

## Current Files:
The current files are silent placeholders. The app will fall back to 
system tones (ToneGenerator) if these files fail to play.

## Tips:
- Keep sound files short (1-3 seconds max)
- MP3 format is recommended
- Keep file sizes small (<100KB each) for fast loading
